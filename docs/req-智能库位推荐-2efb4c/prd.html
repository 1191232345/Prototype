1. 要解决什么问题

在大仓、高货架场景下（部分仓库超 50 万平方英尺，货架如 A08/NGj06 达 34 尺/6 层），库位分配依赖人工经验，导致上架/拣货无效行走多、高频品高位作业、巷道负载失衡。系统需在入库上架时，综合动销、关联、物理属性与库区约束，给出可解释的最优库位，使整条出库动线受益，而非仅优化上架环节。

1.1 核心痛点

痛点 表现 提出场景
动线过长 上架/拣货频繁对角穿越 仓库经理
高度错配 高频 SKU 在高层/远端，6 层 vs 1 层效率差异大 现场运营
负载失衡 部分巷道过载 2 倍，部分闲置 仓库经理
一单分散 一单多件 ABCD 货物跨区存放，拣货往返多 现场运营
业务缺口 VC/SOP/快转无差异化策略 Tommy 总

1.2 产品目标（拆分度量）

目标维度 现状 → 目标 主要受益角色
上架动线 随机分配 → 距入库点/月台路径最短 上架人员
拣货高度 高频占高层 → ≥80% A 类在 1–3 层（以 WMS 层定义为准） 拣货人员
同单聚合 跨区分散 → 高关联 SKU 同巷道/相邻库位 拣货人员
巷道均衡 局部过载 → 各区作业量标准差 ≤20% 仓库经理
空间利用 约 65% → 75% 仓库经理

1.3 MVP 边界

范围 一期（MVP） 二期 三期
入库推荐 B2C 标准上架推荐 VC/SOP 专项策略 多仓自适应
算法能力 动销+同单聚合+品类关联+约束校验 再槽位建议+审批流 快转越库全自动
策略预留 必须预留 VC/SOP/快转策略插槽（默认兜底） 规则上线 —
可视化 Top3 推荐+理由标签；3D 打托（增强） 库位健康度热力图 —

一期默认仅接入 B2C 入库单；VC/SOP/快转通过策略接口占位，避免模型架构返工（响应 Tommy 总反馈）。

2. 用户与主流程

角色 痛点/诉求 关键操作
上架人员 找库位靠经验；不知空位在哪 扫入库单 → Top3 推荐+理由 → 执行/选手动
拣货人员 路径长、爬高多 被动受益：高频品被预置至优位
仓库经理 缺数据、移库难量化 看负载报表 → 审核再槽位建议（二期）
IT/算法 规则变更多、难调试 权重配置、日志、数据质量监控

主流程: 扫入库单 → 识别业务类型(B2C/VC/SOP/快转) → 硬约束过滤 → 综合评分 → Top3+≥2 理由标签 → PDA 执行 → 日志回流（采纳/改选手动+原因）

3. 推荐逻辑

3.1 评分因子

因素 类型 权重 业务含义
动销频率(ABC) 软优化 高 A 类→近出口+低层；C 类→高层/远端
同单共现聚合 软优化 高 同一出库单高频共现 SKU 同巷道/相邻位
品类关联度 软优化 中 购物篮互补品类物理邻近（如牙膏+牙刷）
路径成本 软优化 中 入库点→库位距离，减少对角穿越
库位高度 软优化 中 高频优先 1–3 层，降低高位作业
体积/形状 软优化 中 大重件下层/承重架；异形件专区
巷道负载均衡 软优化 中 超阈值向邻近巷道溢出
容量/承重/安全隔离 硬约束 必须 容积、承重、化学品/食品隔离，不满足则跳过

同单 vs 品类: 优先级为 同出库单共现 > 同巷道 > 同库区；品类关联作次级拉近。

3.2 多业务场景策略（待业务确认）

类型 特征 推荐差异 待确认 来源
B2C 拆零拣选 标准 ABC+关联模型 — 默认
VC 整托/预约制/特定包装 整托区优先；月台时间窗影响紧迫度 周转口径、是否独立库区 Tommy 总
SOP 质检/贴标/组装前置 加工区附近或暂存区，非直接存储区 作业节点、缓冲库位 Tommy 总
快转 入库即出库 越库至发货暂存区；若上架则最易存取位 判定阈值(<24h?)、能否占拣货面 仓库经理

术语待 Super Li 书面确认：VC 可能指 Vendor Central 或供应商直送；SOP 指 Special Operating Procedure（退货翻新、组合包装等）。

3.3 二期：再槽位建议

动销等级变更（如 A→C）或定期扫描时，输出「SKU-X: 当前位 → 推荐位，预计效率提升 Y%」，按 ROI/移库成本排序，每日任务上限+人工审批（避免建议无法落地）。

4. 功能清单

4.1 一期必须

功能 逻辑摘要 验收标准
上架智能推荐 硬约束过滤→综合评分→Top3+理由 响应≤3s；理由≥2；容量校验通过率≥95%
一单多件聚合 近 3 月订单挖掘共现组合→相邻推荐 同单强关联 SKU 同巷道率≥90%

异常兜底 无合适库位→手动选择+必填原因 流程不中断
参数配置 权重可视化调整，实时生效，保留历史 配置即时生效
策略插槽 VC/SOP/快转路由占位+默认 B2C 策略 接口可扩展，不改核心架构

4.2 一期增强（资源允许）

功能 逻辑摘要 验收标准
3D 打托示意 入库单明细→堆码方案→3D 图+稳定性 生成≤2s；托盘利用率≥85%

9. 名词定义

术语 说明
VC 待确认：Vendor Central（平台自营采购）或供应商直送
SOP Special Operating Procedure，特殊流程（质检/贴标/组装/退货翻新等）
快转 入库后极短停留即出库，优先越库
再槽位 基于动销变化的库位动态调整（Re-slotting）
黄金拣货层 人体工学优层，现场对应 WMS 1–3 层（约 0.6–1.5m）
ABC 按出库频率分级：A 高频、C 低频
越库 入库不经存储直接转发出库
